# 🎁 Wish List для Александры

Мини-приложение для координации подарков на день рождения.

## 🚀 Деплой на Railway (5 минут)

### Шаг 1: Создайте репозиторий на GitHub
1. Создайте новый публичный репозиторий (например, `alexandra-wishlist`)
2. Загрузите туда все файлы из этой папки

### Шаг 2: Настройте проект на Railway
1. Зайдите на [railway.app](https://railway.app) → Log in with GitHub
2. Нажмите **New Project** → **Deploy from GitHub repo**
3. Выберите ваш репозиторий `alexandra-wishlist`

### Шаг 3: Добавьте базу данных
1. В проекте нажмите **+** → **New Plugin** → **PostgreSQL**
2. Railway автоматически создаст переменную `DATABASE_URL`
